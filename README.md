# esx-grandmas

## Requirements
- ES_Extended
- mythic_notify
- mythic_progressbars

### Config
check the config to change the values

### Manually
- Download
- Put it in the `resources` directory
- run SQL file for database

## Installation
- run SQL file to put newspaper into database

- Add this in your `server.cfg`:
```
start esx-grandmas
```
